# Pixisphere - Photographer Listing App

A responsive Next.js project featuring a photographer listing platform with advanced search and filter functionality.

---

## Features

- Responsive Navbar with:
  - Logo
  - Search bar (search by name, location, tags)
  - Filter dropdown with:
    - Price slider
    - Rating dropdown (3+, 4+, 4.5+ stars)
    - Styles checkboxes (Traditional, Outdoor, Studio, Candid)
    - Sorting options (Price: low to high, high to low)
- Filters update results dynamically
- Mobile responsive design
- Context API for managing search and filter states
- Uses Tailwind CSS for styling
- Icons from react-icons

---

## Technologies Used

- Next.js (App Router)
- React
- Tailwind CSS
- React Icons

---

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone [https://github.com/yourusername/pixisphere.git](https://github.com/Jainplk/Pixishpere.git)
cd pixisphere

2. Install Dependencies:
npm install
# or
yarn install

3. Run the development server:
npm run dev
# or
yarn dev

4. Open http://localhost:3000 to view it in your browser
